About Smart City Project

The Smart City Project is a web based system which provides information on various types of locations in a number of cities in Australia. The system caters to different types of users such as business, student and tourist users with the system only showing data relevant to you. While the site is early in development, we hope that we will support more cities and locations in the near future.
